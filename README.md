# LuxSmile Dental Boutique — Website

Welcome to your website repository. Everything you need to run and update **luxsmile.ca** is here.

## What's inside

- **`index.html`** — Your website homepage. Edit this to change text, colors, or layout.
- **`public/images/`** — All photos and graphics live here. Add new images here and reference them in `index.html`.
- **`public/js/`** — JavaScript code (currently minimal; we'll add interactivity here as needed).
- **`public/css/`** — Custom stylesheets (if needed in the future; right now Tailwind handles all styling).

## How it works

1. **Edit** — You change the HTML, images, or text.
2. **Verify** — Open `index.html` in your browser to preview changes.
3. **Publish** — Tell me to "go live", "upload", or "publish" — I'll handle sending it to your website automatically.

That's it. No terminals, no commands.
